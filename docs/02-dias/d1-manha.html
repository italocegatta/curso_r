O que é o R?

O R é uma linguagem de programação gratuita para computação estatística e gráficos, mantida pelo R-core team. Com mais de 20.000 pacotes, é uma das ferramentas mais usadas no mundo para análise de dados.

Por que R? Gratuito, comunidade enorme, scripts reprodutíveis, gráficos profissionais.

Instalação

  1. Instale o R em cran.r-project.org
  2. Instale o RStudio em posit.co/download/rstudio-desktop
Warning

Instale o R primeiro, depois o RStudio.
