Manipulação condicional

Muitas vezes você precisa criar colunas cujo valor depende de uma condição. Para isso usamos case_when() (múltiplas condições) e if_else() (condição binária).

case_when() — múltiplas condições

# Classificar filmes por faixa de nota
filmes %>% 
  mutate(
    categoria = case_when(
      nota >= 8.5 ~ "Excelente",
      nota >= 7.0 ~ "Bom",
      nota >= 5.0 ~ "Regular",
      TRUE ~ "Ruim"
    )
  ) %>% 
  count(categoria)
# A tibble: 4 × 2
  categoria     n
  <chr>     <int>
1 Bom        1277
2 Excelente    47
3 Regular    2242
4 Ruim        309
# Mesma lógica aplicada a dados florestais
inv_florestal %>% 
  mutate(
    classe_dap = case_when(
      dap >= 35 ~ "Grossa",
      dap >= 15 ~ "Média",
      TRUE ~ "Fina"
    )
  ) %>% 
  count(classe_dap)
# A tibble: 3 × 2
  classe_dap     n
  <chr>      <int>
1 Fina         232
2 Grossa        35
3 Média        864

if_else() — condição binária

# Classificar como "Sucesso" se faturamento > orçamento
filmes %>% 
  mutate(
    sucesso = if_else(faturamento > orcamento, "Sim", "Não")

Trabalhando com múltiplas tabelas: Joins

Na vida real, os dados raramente estão em uma única tabela. Informações complementares ficam em tabelas separadas que precisam ser cruzadas.

Os tipos de join

left_join()   → Mantém TODAS as linhas da tabela da esquerda
inner_join()  → Mantém APENAS linhas com correspondência nas duas
full_join()   → Mantém TODAS as linhas de ambas
anti_join()   → Mostra o que NÃO tem correspondência (diagnóstico)

Exemplo prático: filmes + elenco

# Carregar tabelas separadas
elenco <- read_csv("../data/elenco.csv")

# left_join: adicionar informações do elenco aos filmes
filmes_com_elenco <- filmes %>% 
  left_join(elenco, by = c("codigo_filme" = "codigo"))

bind_rows() — Empilhando tabelas

# Simulando inventários de anos diferentes
inv_2022 <- inv_florestal %>% mutate(ano_medicao = 2022)
inv_2023 <- inv_florestal %>% mutate(ano_medicao = 2023, dap = dap + rnorm(n(), 2, 1))

inv_historico <- bind_rows(inv_2022, inv_2023)

inv_historico %>% 
  group_by(ano_medicao) %>%
