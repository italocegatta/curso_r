Tidy Data

Dados organizados (tidy data) seguem três regras:

  1. Cada coluna é uma variável
  2. Cada linha é uma observação
  3. Cada célula é um valor

Este conceito, proposto por Hadley Wickham, é a base para todo o tidyverse. Quando seus dados estão organizados, todas as funções funcionam perfeitamente juntas.

Formato largo vs formato longo

Formato largo (comum em planilhas Excel, visualização humana):

parcela dap_2021 dap_2022 dap_2023
1 15.2 17.8 20.1
2 14.8 16.9 19.5

Formato longo (ideal para análise e ggplot2):

parcela ano dap
1 2021 15.2
1 2022 17.8
1 2023 20.1
2 2021 14.8
2 2022 16.9
2 2023 19.5

pivot_longer() — largo → longo

# Simulando inventário contínuo (formato largo)
inv_largo <- tibble(
  parcela = 1:3,
  dap_2021 = c(15.2, 14.8, 13.5),
  dap_2022 = c(17.8, 16.9, 15.2),
  dap_2023 = c(20.1, 19.5, 17.8)
)
inv_largo
# A tibble: 3 × 4
  parcela dap_2021 dap_2022 dap_2023
    <int>    <dbl>    <dbl>    <dbl>
1       1     15.2     17.8     20.1
2       2     14.8     16.9     19.5
3       3     13.5     15.2     17.8
# Converter para formato longo
inv_largo %>% 
  pivot_longer(
    cols = starts_with("dap_"),
    names_to = "ano",
    values_to = "dap",
    names_prefix = "dap_"
  ) %>% 
  mutate(ano = as.numeric(ano))
# A tibble: 9 × 3
  parcela   ano   dap
    <int> <dbl> <dbl>
1       1  2021  15.2
2       1  2022  17.8
3       1  2023  20.1
4       2  2021  14.8
5       2  2022  16.9
6       2  2023  19.5
7       3  2021  13.5
8       3  2022  15.2
9       3  2023  17.8
# Agora podemos analisar e plotar
inv_longo <- inv_largo %>% 
  pivot_longer(starts_with("dap_"), names_to = "ano", values_to = "dap", names_prefix = "dap_") %>% 
  mutate(ano = as.numeric(ano))

inv_longo %>% 
  ggplot(aes(x = ano, y = dap, color = factor(parcela))) +
  geom_line(size = 1) +
  geom_point(size = 2) +
  labs(title = "Evolução do DAP por parcela", x = "Ano", y = "DAP (cm)", color = "Parcela")

pivot_wider() — longo → largo

# Tabela-resumo: volume por parcela × espécie
inv_florestal %>% 
  group_by(parcela, especie) %>%

Separar e unir colunas

# Código da parcela: "T1-P2" → talhão + parcela
tibble(codigo = c("T1-P1", "T1-P2", "T2-P1")) %>% 
  separate(codigo, into = c("talhao", "parcela"), sep = "-")
# A tibble: 3 × 2
  talhao parcela

2 T1     P2

# Unir colunas
tibble(talhao = "T1", parcela = "P1") %>% 
  unite("codigo", talhao, parcela, sep = "-")
