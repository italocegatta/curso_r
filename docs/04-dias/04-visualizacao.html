Visualização de dados com ggplot2

Dia 2 — Tarde | 4 horas

Objetivos

Ao final deste capítulo, você será capaz de:

  • Entender a gramática de gráficos do ggplot2

  • Criar visualizações profissionais com customização completa

  • Usar facetas para comparar grupos

  • Exportar gráficos em alta resolução

A gramática de gráficos

O ggplot2 é baseado na Gramática de Gráficos (Grammar of Graphics). Todo gráfico é construído por camadas:

ggplot(dados, aes(x, y)) +   # dados + mapeamento estético
  geom_tipo() +               # geometria (o que desenhar)
  scale_tipo() +              # escalas (cores, eixos)
  theme_tipo() +              # tema (aparência)
  labs(...)                   # rótulos (títulos)

Vamos construir um gráfico passo a passo, adicionando camadas:

# Passo 1: dados + eixos (tela vazia)
inv_florestal %>%
  ggplot(aes(x = dap, y = altura))
# Passo 2: geometria de pontos
inv_florestal %>%
  ggplot(aes(x = dap, y = altura)) +
  geom_point()
# Passo 3: cor por espécie
inv_florestal %>%
  ggplot(aes(x = dap, y = altura, color = especie)) +
  geom_point(alpha = 0.5)
# Passo 4: linha de tendência + tema + rótulos
inv_florestal %>%
  ggplot(aes(x = dap, y = altura, color = especie)) +
  geom_point(alpha = 0.5) +
  geom_smooth(method = "lm", se = FALSE) +
  labs(
    title = "Relação hipsométrica",
    subtitle = "DAP × Altura por espécie",
    x = "DAP (cm)",
    y = "Altura (m)",
    color = "Espécie"
  ) +
  theme_minimal()

Geometrias essenciais

Uma variável contínua: histograma e densidade

# Distribuição diamétrica — histograma
inv_florestal %>%
  ggplot(aes(x = dap)) +
  geom_histogram(bins = 30, fill = "#18BC9C", color = "white") +
  labs(title = "Distribuição diamétrica", x = "DAP (cm)", y = "Frequência")
# Densidade — mais suave que histograma, bom para comparar grupos
inv_florestal %>%
  ggplot(aes(x = dap, fill = especie)) +
  geom_density(alpha = 0.5) +
  labs(title = "Distribuição diamétrica por espécie", x = "DAP (cm)", y = "Densidade")

Duas variáveis: dispersão, linha, boxplot

# Dispersão com tendência
inv_florestal %>%
  ggplot(aes(x = dap, y = volume)) +
  geom_point(alpha = 0.3, color = "#2C3E50") +
  geom_smooth(method = "lm", color = "#E74C3C") +
  labs(title = "Relação DAP × Volume", x = "DAP (cm)", y = expression(Volume~(m^3)))
# Boxplot — comparar distribuições
inv_florestal %>%
  ggplot(aes(x = especie, y = volume, fill = especie)) +
  geom_boxplot() +
  labs(title = "Volume por espécie", x = NULL, y = expression(Volume~(m^3))) +
  theme(legend.position = "none")
# Volume total por parcela (barras)
inv_florestal %>%
  group_by(parcela) %>%
  summarise(volume_total = sum(volume)) %>%
  ggplot(aes(x = reorder(factor(parcela), volume_total), y = volume_total)) +
  geom_col(fill = "#18BC9C") +
  coord_flip() +
  labs(title = "Volume total por parcela", x = "Parcela", y = expression(Volume~(m^3)))

Customização de cores

# Cores pré-definidas (ColorBrewer)
inv_florestal %>%
  ggplot(aes(x = dap, y = altura, color = especie)) +
  geom_point(alpha = 0.5) +
  scale_color_brewer(palette = "Set1") +
  labs(color = "Espécie")
# Escala viridis (acessível para daltônicos)
inv_florestal %>%
  ggplot(aes(x = dap, y = altura, color = volume)) +
  geom_point(alpha = 0.7) +
  scale_color_viridis_c(option = "plasma") +
  labs(color = expression(Volume~(m^3)))

Facetas: subgráficos por grupo

# Um gráfico por espécie
inv_florestal %>%
  ggplot(aes(x = dap, y = altura)) +
  geom_point(alpha = 0.5, color = "#2C3E50") +
  geom_smooth(method = "lm", se = FALSE, color = "#E74C3C") +
  facet_wrap(~especie) +
  labs(title = "Relação hipsométrica por espécie", x = "DAP (cm)", y = "Altura (m)")
# Grade por espécie × parcela (selecionando 6 parcelas)
inv_florestal %>%
  filter(parcela %in% c(1, 2, 3, 7, 12, 15)) %>%
  ggplot(aes(x = dap, fill = especie)) +
  geom_histogram(bins = 20, alpha = 0.7, position = "identity") +
  facet_grid(parcela ~ especie) +
  labs(title = "Distribuição diamétrica — parcelas selecionadas")

Temas e aparência final

# Comparação de temas
p <- inv_florestal %>%
  ggplot(aes(x = dap, y = altura, color = especie)) +
  geom_point(alpha = 0.5)

# Cinco temas diferentes
p + theme_minimal() + labs(title = "theme_minimal()")
p + theme_bw() + labs(title = "theme_bw()")
p + theme_light() + labs(title = "theme_light()")
p + theme_classic() + labs(title = "theme_classic()")
p + theme_dark() + labs(title = "theme_dark()")
# Gráfico pronto para relatório
inv_florestal %>%
  ggplot(aes(x = dap, y = altura, color = especie)) +
  geom_point(alpha = 0.5, size = 2) +
  geom_smooth(method = "lm", se = TRUE, alpha = 0.2) +
  scale_color_brewer(palette = "Set1") +
  labs(
    title = "Relação hipsométrica — Inventário Florestal",
    subtitle = "Curvas ajustadas por espécie com intervalo de confiança de 95%",
    x = "DAP (cm)",
    y = "Altura total (m)",
    color = "Espécie",
    caption = "Fonte: Dados do inventário florestal"
  ) +
  theme_minimal(base_size = 13) +
  theme(
    plot.title = element_text(face = "bold"),
    legend.position = "bottom"
  )

TipFormatos e resolução
  • PNG — melhor para web e apresentações
  • PDF — melhor para publicações e impressão
  • SVG — gráficos vetoriais redimensionáveis
  • DPI — 150 para web, 300 para impressão, 600 para publicação

Dashboard: combinando gráficos com patchwork

# Quatro gráficos em um painel
p1 <- inv_florestal %>%
  ggplot(aes(x = dap)) + geom_histogram(bins = 30, fill = "#18BC9C") + labs(title = "Distribuição diamétrica")

p2 <- inv_florestal %>%
  ggplot(aes(x = dap, y = altura, color = especie)) + geom_point(alpha = 0.5) + labs(title = "Relação hipsométrica")

p3 <- inv_florestal %>%
  ggplot(aes(x = especie, y = volume, fill = especie)) + geom_boxplot() + labs(title = "Volume por espécie") + theme(legend.position = "none")

p4 <- inv_florestal %>%
  group_by(parcela) %>% summarise(v = sum(volume)) %>%
  ggplot(aes(x = factor(parcela), y = v)) + geom_col(fill = "#2C3E50") + labs(title = "Volume por parcela", x = "Parcela")

(p1 + p2) / (p3 + p4) +
  plot_annotation(title = "Dashboard — Inventário Florestal", theme = theme(plot.title = element_text(face = "bold", size = 16)))

Exercícios

Exercício 1

Crie um histograma da distribuição diamétrica (dap) com 30 bins, colorido em verde (#18BC9C). Adicione título e rótulos aos eixos.

Resultado esperado:

Histograma com pico em torno de 15-20 cm, distribuição assimétrica à direita.

Exercício 2

Crie um boxplot de volume por especie. Qual espécie tem maior variabilidade?

Resultado esperado:

Eucalipto com caixa mais larga e mediana mais alta que Pinus.

Exercício 3

Plote dap × volume com pontos (alpha = 0.3) e uma curva de tendência suave (geom_smooth). Colora os pontos por espécie.

Resultado esperado:

Nuvem de pontos com duas curvas não-lineares — volume cresce exponencialmente com DAP.

Exercício 4

Usando facet_wrap(~especie), crie um gráfico de dispersão dap × altura com um painel para cada espécie.

Resultado esperado:

Dois painéis lado a lado, cada um com a relação hipsométrica de uma espécie.

Exercício 5

Crie um gráfico de barras do volume total por parcela (top 10), ordenado do maior para o menor. Use coord_flip() para facilitar a leitura.

Resultado esperado:

Gráfico horizontal com 10 barras verdes, parcela 7 com maior volume.

Exercício 6

Exporte seu melhor gráfico como PNG com 8 polegadas de largura, 5 de altura e 300 DPI.
